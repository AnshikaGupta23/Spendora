💸 Spendora – Expense Tracker Web App

A modern and intuitive expense tracking application to help users manage finances, track spending habits, and make smarter financial decisions.

🚀 Overview

Spendora is a full-stack web application designed to simplify personal finance management. It allows users to track income and expenses, categorize transactions, and visualize financial data through an interactive dashboard.

The goal of this project is to provide a clean, efficient, and user-friendly solution for budgeting and expense monitoring.

✨ Features
📊 Dashboard Analytics – Visualize your spending patterns
💰 Expense Tracking – Add, edit, and delete transactions easily
📁 Category Management – Organize expenses by category
📅 Time-based Filtering – View daily, monthly, or yearly expenses
🔍 Search & Filter – Quickly find transactions
📊 Advanced analytics & charts
📈 Financial Insights – Understand your spending behavior
🔐 Secure Backend – Data handled safely via server APIs

⚙️ Installation & Setup
1️⃣ Clone the repository
git clone https://github.com/AnshikaGupta23/Spendora.git
cd Spendora
2️⃣ Install dependencies
For backend:
cd server
npm install
For frontend:
cd client
npm install
3️⃣ Setup environment variables

Create a .env file inside the server folder:

PORT=5000
MONGO_URI=your_mongodb_connection_string
4️⃣ Run the project
Start backend:
cd server
npm run dev
Start frontend:
cd client
npm run dev
